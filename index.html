Design for the second year

The first data source is always fine. I build configuration-driven Medallion frameworks — generic Bronze, Silver, and Gold layers — so source fifteen ingests in hours, not weeks. The architecture pays compound interest.

Governance is not a phase

Lineage and access control bolted on after launch aren't governance — they're panic. I wire Unity Catalog constraints and Delta Lake schema enforcement into the design from day one, so the auditor's questions are already answered.

Automate the repetition

Claude Code runs inside VS Code throughout the pipeline loop — scaffolding PySpark notebooks, generating SQL transforms, reviewing pull requests for drift. Engineers spend time on data modeling, not rewriting the same ingestion boilerplate for the twelfth time.
